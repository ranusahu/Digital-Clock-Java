# Digital Clock (Java)

This is a simple Java console-based digital clock application.

## Features
- Displays current system time
- Time format: HH:MM:SS
- Updates time every second

## Concepts Used
- Thread and sleep
- Date and Time
- Infinite loop

## How to Run
1. Compile the program:
   javac DigitalClock.java
2. Run the program:
   java DigitalClock
